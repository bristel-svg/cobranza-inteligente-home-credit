# Datos

Este proyecto utiliza el dataset público Home Credit Default Risk.

Por tamaño y licencia, los archivos crudos no se incluyen en el repositorio.
Descargar desde Kaggle y ubicar los CSV en:

data/raw/

Archivos mínimos requeridos para la demo principal:

- application_train.csv
- previous_application.csv
- installments_payments.csv

Archivos opcionales:

- bureau.csv
- bureau_balance.csv
- POS_CASH_balance.csv
- credit_card_balance.csv
